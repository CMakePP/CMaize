cmaize.user_api/cmaize_project

cmaize.user_api/cmaize_project.cmaize_project(_cp_name)

Note

This is a macro, and so does not introduce a new scope.

Creates a CMaize project and the underlying CMake project.

Parameters:

_cp_name (desc) – Name for the project, must be unique and conform to rules of a CMake project name.
