cmaize.utilities/split_version.cmake¶

cmaize.utilities/split_version.cmake.cmaize_split_version(major minor patch tweak version)¶

Split the given version string into its components. Some return values may be blank.

Parameters

  • major (str) – Returned first component of the version.

  • minor (str) – Returned second component of the version.

  • patch (str) – Returned third component of the version.

  • tweak (str) – Returned fourth component of the version.

  • version (str) – Version string to be separated into components.

Returns

major will be set to the first component of version.

Return type

str

Returns

minor will be set to the second component of version.

Return type

str

Returns

patch will be set to the third component of version.

Return type

str

Returns

tweak will be set to the fourth component of version.

Return type

str
